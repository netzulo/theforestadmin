theforestadmin
==============

Command line tool written on python to setup TheForest Servers


Setup
*****

```python setup.py install```


Tests
*****

```python setup.py test```


Server Usage
************

::
    
    server.py --help


Deploy Usage
************

::
    
    deploy.py --help
